# Dangerous Mixture Checker

A simple web application built with **HTML, CSS, and JavaScript** that helps users check if mixing two household cleaning products may create a dangerous chemical reaction.

## 🚨 Purpose

Some cleaning products should never be mixed because they can release toxic gases or harmful substances.

This project allows the user to select two products and instantly see whether the combination is dangerous.

## 🧪 Included Products

- Bleach
- Vinegar
- Ammonia
- Alcohol

## ⚙️ Features

- Two dropdown menus to choose products
- Verification button disabled until both products are selected
- Detects dangerous combinations
- Warns the user with a clear message
- Prevents selecting the same product as a valid mixture
- Simple and beginner-friendly code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📚 What I Practiced

- DOM selection (`getElementById`)
- Event listeners
- Functions
- Arrays and `.some()`
- Conditional logic
- Dynamic form validation
- Updating text content with JavaScript

## 🚀 How to Use

1. Open `index.html`
2. Select two products
3. Click **Verify**
4. Read the result message
